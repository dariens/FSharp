Fibonacci in F#

The following is a simple recursive function to calculate the fibonacci of a particular value:

let rec fib n =
    match n with
    | 0 -> 0
    | 1 -> 1
    | n -> n + fib(n-1)

To use this function you simply pass an integer to the function. Follows an example which creates a variable fib5 and binds it to the value created by passing 5 to the fib function

let fib5 = fib 5

The variable fib5 now contains the follwing value:

15
